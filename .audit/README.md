# Audit Templates

Use these templates when assigning an auditor session.

- `security.md`
- `robustness.md`
- `code-quality.md`
- `technical-debt.md`
- `test-coverage.md`

Recommended workflow:
1. Copy the relevant template into the auditor context.
2. Add concrete assignment details (scope, branch/tag, deadline).
3. Require ranked findings and explicit file/line references.
